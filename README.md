# 投资机构数据库

中国头部股权投资机构数据库，收录 2020–2025 年清科、投中、融中财经、36氪、投资界、母基金周刊等权威榜单数据。

## 数据概览

| 指标 | 数值 |
|------|------|
| 机构数量 | 424家 |
| 标签总数 | 1262条 |
| 年份覆盖 | 2020–2025年 |
| 榜单来源 | 7个 |

## 查看数据库

直接在浏览器打开 `index.html` 即可浏览、搜索、筛选机构数据。

## 数据结构

每家机构包含：
- `name` — 机构名称
- `type` — 机构类型（VC/PE/早期/国资/CVC等）
- `fund_scale` — 基金管理规模
- `focus_areas` — 关注领域
- `cases` — 代表性投资案例
- `first_year` / `last_year` — 最早/最近上榜年份
- `years_active` — 上榜活跃年数
- `source_orgs` — 上榜来源
- `tag_count` — 标签数量
